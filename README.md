
# 📘 Numerical Methods Lab (BSc CSIT 3rd Semester)

This repository contains C programs for Numerical Methods laboratory works of BSc CSIT (2nd Year, 3rd Semester).


## 🎯 Purpose
- To implement Numerical Methods algorithms in C
- To understand theoretical concepts through practical coding
- For academic learning and reference

## 🛠️ Tools Used
- Language: C
- IDE: VS Code


## 🚀 How to Run

- Clone the repository
- git clone
```
      https://github.com/josephmaharjan/numerical-methods-lab-works.git 
```
- Open the folder in VS Code
- Compile and run any C program:
```
    gcc filename.c -o output
    ./output 
```
